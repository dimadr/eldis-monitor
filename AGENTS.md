# AGENTS — правила работы агента

## Главное правило
**Ни одного действия без недвусмысленного «делай» от пользователя.**

Любая команда, создание/изменение/удаление файлов, запуск сервера, curl, pip — только после чёткого подтверждения.

## Порядок работы

1. Пользователь даёт задачу или план
2. Агент отвечает планом или уточнением
3. Агент ждёт «делай» или «да»
4. Только после этого — выполнение одного шага
5. После выполнения — ждать следующего указания

**Запрещено:**
- Выполнять несколько шагов подряд без подтверждения
- Додумывать за пользователя и делать по своему усмотрению
- Создавать/редактировать/удалять файлы без команды
- Запускать сервер или тестировать API без разрешения
- Писать команды в плане (только текст на русском)

## Создание архивов

Для создания архива проекта использовать (только по команде):
```bash
cd /root/eldis && rm -f NAME.tar.gz && tar -czhf NAME.tar.gz eldis-monitor/
```

## Чистка приватных данных для GitHub
Все изменения приватности — только в копии в директории GitHub.
Исходный проект `/root/eldis/` не трогать никогда.
