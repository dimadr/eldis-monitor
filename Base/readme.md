# База знаний

В этой директории вы можете хранить руководства по эксплуатации приборов учёта в формате PDF, а также любую другую документацию, которая используется при расшифровке и анализе нештатных ситуаций (НС).

Структура директории свободная — вы можете организовать файлы по производителям, типам приборов или любым другим удобным способом.

## Для чего это нужно

- При генерации отчётов система может ссылаться на конкретные руководства для пояснения кодов ошибок и нештатных ситуаций.
- PDF-документы из этой директории могут быть прикреплены к отчётам или использованы сервисными инженерами для диагностики.
- Текстовые и markdown-файлы могут содержать логику анализа, описание кодов НС и другие вспомогательные материалы.
